*Formula for an exciting, non-slaveic, heedful life:*

#+begin_quote
1. Habit Tracking (Daily)
   1. Work Intervals (Pomodoro)
   2. Exercise (Cardio)
   3. Stretching (Yoga)
   4. Cold Water (Showers)
   5. Low EMF Infrared Sauna (30 min)
   6. Meditation (Vipassana, sitting [[https://en.wikipedia.org/wiki/Seiza][seiza]] bench)
   7. Some deliberately creative, artistic, challenging, and stimulating activity (music/art/coding, etc)
2. Time Blocking
3. Task Tracking
4. Journal
   1. Gratitude
   2. Reflection
5. Goal/Vision Setting
6. [[https://en.wikipedia.org/wiki/Mastermind_group][Mastermind Group]]
7. A note-taking, organizational system ([[https://github.com/vicrdguez/dendroam][dendroam]])
8. Only exceptions to the above, are if you're on [[https://kunja.dhamma.org/][retreat]], but hold to as much as you can.
#+end_quote
